**Bridge** is a "Human-in-the-Loop" workflow tool that connects Google Gemini (Web) directly to your local development environment. 

It allows you to code inside the browser, sync files to your local machine, execute terminal commands, and‚Äîmost importantly‚Äî**automatically feed execution errors back to Gemini** for autonomous self-correction.

## ‚ú® Features

* **üìÇ File Syncing:** Automatically detects `## FILE:` blocks in Gemini's response and saves them to your disk.
* **üíª Command Execution:** Detects `## CMD:` blocks and runs them in your local terminal.
* **üîÑ The "Auto-Loop" (Self-Healing):**
    * If a command fails (non-zero exit code), the tool captures the error log.
    * The extension automatically types the error back into the Gemini chat.
    * Gemini analyzes the error and generates a fix, restarting the cycle.
* **üéõÔ∏è Overlay UI:** A draggable, "Always-on-Top" widget to trigger manual syncs without leaving the browser.

## üõ†Ô∏è Installation

### 1. Python Server (The Backend)
1.  Clone this repository.
2.  Install the required Python dependencies:
    ```bash
    pip install flask pyautogui pygetwindow
    ```
3.  Open `server.py` and configure your **Project Root**:
    ```python
    # Inside server.py
    PROJECT_ROOT = r"C:\Path\To\Your\Project" 
    ```
4.  Start the server:
    ```bash
    python server.py
    ```

### 2. Chrome Extension (The Frontend)
1.  Open Chrome and navigate to `chrome://extensions/`.
2.  Enable **Developer mode** (top right toggle).
3.  Click **Load unpacked**.
4.  Select the folder containing `manifest.json`.
